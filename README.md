# NH투자증권 사건 — 인포그래픽 자료 아카이브

GitHub Pages 게시용 정적 사이트입니다. `index.html`이 전체 목록 페이지입니다.

## 게시 방법 (kim-sungman/touchi.github.io 기준)
1. 이 폴더 안의 **모든 파일(index.html + 01~12 + README.md)** 을 저장소에 업로드(또는 드래그) 후 Commit.
2. GitHub 저장소 → **Settings → Pages** → Source: `Deploy from a branch` → Branch: `main` / 폴더 `/(root)` → Save.
3. 1~2분 뒤 게시 주소: **https://kim-sungman.github.io/touchi.github.io/**
   - 목록 페이지: 위 주소
   - 개별 자료: 위 주소 + 파일명 (예: `.../touchi.github.io/01-org-chart-court-gov.html`)

## 노션(oopy) 임베드
- 노션 페이지에서 `/embed` → 위 게시 주소 붙여넣기
